JSON format

Savollarni quyidagi formatda JSON fayliga yozing va import qiling. Bir savol yoki massiv (ko'p savollar) yuklash mumkin. 

[
  {
    "Savol": "Parametr $a$ ning
      qanday qiymatlarida
      $|ax - 2| < 4$
      tengsizligining yechim
      to'plami bo'sh emas?",
    "Fan": "Matematika",
    "Mavzu": "Chiziqli tengsizliklar",
    "Qiyinlik": "hard",
    "A": "Barcha $a \\neq 0$ uchun",
    "B": "$a > 0$ faqat",
    "C": "$|a| \\geq 1$ faqat",
    "D": "$a < 0$ faqat",
    "Togri_javob": "A",
    "Izoh": "Izohlash...",
    "Yil": 2024,
    "Manba": "DTM 2024"
  }
]
Majburiy maydonlar:
Savol — Savol matni (LaTeX qo'llab-quvvatlanadi)
Fan — Matematika / Ingliz tili / Reading / Boshqa
Mavzu — Savol mavzusi
Qiyinlik — easy / medium / hard (yoki oson/o'rta/qiyin)
A, B, C, D — Javob variantlari
Togri_javob — A, B, C yoki D

Ixtiyoriy maydonlar:
Izoh — To'g'ri javob izohi
Yil — Yil (masalan: 2024)
Manba — Manba nomi
Eslatmalar
  • LaTeX formulalar: $x^2$ yoki \(x^2\)
  • Massiv yoki bitta obyekt yuklash mumkin
  • Qiyinlik: easy, medium, hard
  • To'g'ri javob: faqat A, B, C yoki D
  • Fayl hajmi: cheksiz (bir vaqtda ko'p savol)
